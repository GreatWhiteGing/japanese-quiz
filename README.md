# Japanese Quiz App

A Python application to help learn Japanese hiragana and katakana through interactive quizzing. Available as both a console app and a GUI application

## Features

- Quiz yourself on hiragana, katakana, or both
- Tracks your score and accuracy percentage per session
- Saves session history between uses
- Displays last 5 sessions on startup
- Results screen showing incorrect answers to revisit
- Available as both a console app and a GUI application

## Technologies Used

- Python 3
- tkinter (GUI)
- JSON (data persistence)
- Git / GitHub (version control)

## How to Run

1. Clone the repository
2. Make sure Python 3 is installed
3. Run the console version: `python quiz.py`
4. Run the GUI version: `python app.py`

## Project Structure

- `app.py` - GUI application
- `quiz.py` - Console application
- `data.py` - Hiragana and katakana data
- `score.py` - Score saving and history functions
